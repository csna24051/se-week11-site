# 美容診断アプリ

## アプリ概要（主要機能）
このアプリは、顔の形や肌タイプ、髪の色など10項目の入力から、パーソナルカラーや顔タイプを診断するWebアプリです。
診断結果として、おすすめの髪型、服装、似合う色、アドバイスを表示します。

## 対象ユーザ
美容診断をしたい人

## 目的
一人一人に合うものの情報をその人自身で確かめる。

## 起動方法
1. プロジェクトをダウンロードする。
2. index.html をブラウザで開く。
3. 10項目を入力する。
4. 「診断する」を押すと診断結果が表示される。
5. 「リセット」を押すと入力内容と診断結果を初期状態に戻せる。
